# House-of-Data

Project Website: https://ryannwongg.github.io/House-of-Data/

ScreenCast Video: https://www.youtube.com/watch?v=9bPUNGVcRh4

Libraries: 

NBA API - Data Sourcing for visualizations


Code: 

The data directory contains all the statistics and advanced statistics we used for each season from 2005-2006 to 2024-2025 as well as the statistics of each team that we compared from the 2000 season and the 2025 season.

The html contains the structure of the webpage and the css contains the styling

pace.js: Represents the pace visualization (the first one)

comparison.js: Represents the comparison of top players visualization (the second one)

threepoint.js: Represents the threepoint chart visualization (the third one)

defense.js: Represents the defense visualization (the fourth one)

shot_chart.js: Represents the 3D visualization of the NBA court comparing shots between Michael Jordan and Lebron James. (the last one)
